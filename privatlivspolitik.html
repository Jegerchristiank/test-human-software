Privatlivspolitik

Vi passer på dine data og bruger kun det, der er nødvendigt for at levere Human Biologi Studio.

Tilbage
Vilkår Privatlivspolitik Persondatapolitik Handelsbetingelser

1. Hvilke data vi indsamler

  • Kontodata: e-mailadresse, navn og logininformation.
  • Profil og indstillinger: plan, præferencer og eget nøglevalg.
  • Brugsdata: runder, score, historik og læringsprogression.
  • Indhold du sender: svar, noter, uploads og eventuelle skitser.
  • Tekniske data: logdata, fejl og enhedsinformation.

2. Formål

  • Levere og forbedre tjenesten, herunder indlæsning af spørgsmål og historik.
  • Drive sikker login, betaling og adgangskontrol.
  • Yde support og fejlretning ved behov.

3. Retsgrundlag

Vi behandler dine data, fordi det er nødvendigt for at levere tjenesten (aftale), for at sikre stabil drift (legitim interesse) og når loven kræver samtykke.

4. Deling med tredjeparter

Vi deler data med betroede leverandører, der driver hosting, login, betaling og AI-funktioner. Vi sælger aldrig dine data.

5. Opbevaring

Data opbevares, så længe din konto er aktiv, eller indtil du beder om sletning. Visse logdata kan gemmes længere af sikkerhedshensyn.

6. Dine rettigheder

  • Få indsigt i dine data.
  • Få rettet eller slettet data.
  • Få udleveret dine data i et maskinlæsbart format.

Du kan håndtere disse rettigheder via din konto i appen.

7. Sikkerhed

Vi bruger adgangskontrol, krypteret kommunikation og overvågning for at beskytte dine data.

8. Kontakt

Ved spørgsmål til privatliv kan du kontakte os via kristenson.dk eller via din konto-side i appen.
